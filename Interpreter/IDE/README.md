## 버전
 - 아마도 0.1일 것입니다

## 특징
 - 코드가 더럽습니다.
 - 많이 허접합니다.
 - Java로 짜여진 IDE입니다.
 - 시뮬레이터로 작동시키는 기능이 존재합니다.
 - 언어의 사용에 필요한 특수문자를 모두 단축키로 지정해두었습니다.
 - 사용한 폰트는 Monospace입니다.
